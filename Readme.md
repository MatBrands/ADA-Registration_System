## Execução
Como a proposta do projeto foi a utilização de técnicas e tecnologias vistas em aula, não foi necessário o uso de pacotes externos

- Navegue até a pasta destino
```sh
cd registration_system
```

- Execute o programa
```sh
python __init__.py
```
